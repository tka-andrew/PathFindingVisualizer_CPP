# Path Finding Visualizer in CPP
This is mainly to visualize the path finding algorithm written in CPP.

## DEPLOYMENT MODE

### To build this image
```
sudo docker build -t tka-andrew/path-finding-visualizer:latest .
```

### To run the docker image
```
./runDockerFile.sh
```
